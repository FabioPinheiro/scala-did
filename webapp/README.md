# WEBAPP module

## build and run

[See info about who to buind and run in demo](./../demo/README.md)

**Run Frontend:**

```shell
# Live reload
sbt '~serviceworker/fastLinkJS';
sbt '~webapp/fastLinkJS' # run on another console
npm run dev
```
